# Handoff: SAM 2 Video Labeling Tool (auto_labelling)

**Date:** 2026-05-27
**Repo:** `https://github.com/patdev495/sam2_labelling.git` (branch: `master`)

## What this project is

A PySide6 desktop app that uses **SAM 2** (Meta's Segment Anything Model) to accelerate bounding-box labeling for YOLO object detection training. Target use case: warehouse door camera, person detection/tracking.

## How to run

```bash
# Base deps
uv sync

# ML deps (only when ready to test SAM 2)
uv sync --extra ml
pip install git+https://github.com/facebookresearch/sam2.git

# Run
uv run python -m auto_labelling.main
# or double-click: run.bat
```

## Workflow (design intent)

1. Open video ‚Üí Extract frames (configurable skip)
2. Add tracks (click or box prompt on a person)
3. Auto Label (SAM 2 propagate masks ‚Üí convert to boxes)
4. Review timeline, correct bad boxes manually
5. Export YOLO `.txt` files (one per frame, same folder)
